## การจำกัดการเข้าถึงข้อมูล (RBAC)

### Student (นักศึกษา)
- ✅ เห็นเฉพาะโปรเจกต์ที่ตัวเองเป็นสมาชิก
- ✅ เห็นเฉพาะ Coaching Sessions ที่ตัวเองเข้าร่วม
- ✅ เห็นเฉพาะ Action Items ที่ถูกมอบหมายให้ตัวเอง
- ❌ ไม่เห็นข้อมูลของนักศึกษาคนอื่น
- ❌ ไม่เห็นข้อมูลของอาจารย์คนอื่น

### Advisor (อาจารย์ที่ปรึกษา)
- ✅ เห็นโปรเจกต์ที่ตัวเองเป็นที่ปรึกษา
- ✅ เห็น Coaching Sessions ที่ตัวเองเป็นผู้ดูแล
- ✅ เห็นข้อมูลของนักศึกษาที่อยู่ในความดูแล
- ❌ ไม่เห็นข้อมูลของนักศึกษาที่ไม่ได้ดูแล
- ❌ ไม่เห็นข้อมูลของอาจารย์คนอื่น

### Admin (ผู้บริหาร)
- ✅ เห็นข้อมูลทั้งหมดในระบบ
- ✅ เห็นโปรเจกต์ทั้งหมด
- ✅ เห็น Coaching Sessions ทั้งหมด
- ✅ เห็นข้อมูลผู้ใช้ทั้งหมด
- ✅ เข้าถึง Audit Logs

### Committee (กรรมการภายนอก)
- ✅ เห็นเฉพาะโปรเจกต์ที่ได้รับมอบหมายให้ประเมิน
- ✅ เห็นเฉพาะ Evaluations ที่ตัวเองทำ
- ❌ ไม่เห็นข้อมูลอื่นๆ ที่ไม่เกี่ยวข้อง

## วิธีใช้งาน

1. เปิดหน้าเว็บ
2. กรอก Email และ Password
3. คลิก "เข้าสู่ระบบ"
4. ระบบจะแสดง Dashboard ตาม Role ของคุณ
5. ข้อมูลที่แสดงจะถูกจำกัดตามสิทธิ์การเข้าถึง

## หมายเหตุ

- ระบบนี้เป็น Demo Version สำหรับการทดสอบ
- ใน Production ต้องเชื่อมต่อกับ Backend API จริง
- Password จะถูก hash ด้วย bcrypt/Argon2 ที่ Backend
- JWT tokens จะถูกจัดการที่ Backend
